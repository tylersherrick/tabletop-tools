# Tabletop Tools

Tabletop RPGs is a genre of game where players collaboratively tell stories and
go on adventures. Virtual tabletops (VTTs) are digital tools designed to make
these games easier to play, especially with friends online. Let's build some
common VTT features!

## Instructions

Complete the body of each function in `index.js` according to the JSDoc.

Live serve `index.html` and use the browser console to test your code.

>
> Make a commit after completing each function to practice using git!
